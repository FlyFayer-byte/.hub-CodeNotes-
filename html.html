Основи HTML

Що таке HTML?

HTML (HyperText Markup Language) – це мова розмітки гіпертексту, яка використовується для створення веб-сторінок. Вона визначає структуру сторінки за допомогою спеціальних тегів.

Основні теги HTML

HTML складається з тегів, які розміщуються в кутових дужках < >.

Наприклад:

<p>Це параграф</p>

Більшість тегів мають відкриваючий <p> і закриваючий </p>.

Ось мінімальна структура HTML-документа:

<!DOCTYPE html>
<html>
  <head>
    <title>Моя перша веб-сторінка</title>
  </head>
  <body>
    <h1>Привіт, світ!</h1>
    <p>Це мій перший HTML-документ.</p>
  </body>
</html>

Опис основних тегів:

  • <html> – огортає весь HTML-код.
  • <head> – містить метаінформацію (назву сторінки, стилі тощо).
  • <title> – задає заголовок сторінки у вкладці браузера.
  • <body> – містить весь контент, який бачить користувач.
  • <h1> ... <h6> – використовуються для заголовків на сторінці від найбільшого <h1> до найменшого <h6>.
  • <p> – параграф тексту.
  • <a href="URL"> – гіперпосилання.
  • <img src="URL" alt="Опис зображення"> – зображення.
  • <ul>, <ol>, <li> – списки (маркерований, нумерований, елемент списку).

Тестування

Що таке HTML і для чого він використовується?

HTML – це мова розмітки для створення веб-сторінок.

Які теги використовується для заголовків на сторінці?

Заголовки на сторінці створюються тегами <h1> ... <h6>

Який тег потрібен для створення посилання?

Тег <a href="URL"> використовується для створення посилання.

Що робить тег <p>?

Тег <p> створює параграф тексту.

Як додати зображення на сторінку? Напиши приклад коду.

За допомогою коду <img src="URL" alt="Опис зображення"> можна додати зображення на сторінку.
