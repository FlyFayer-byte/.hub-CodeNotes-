Основи JavaScript

Що таке JavaScript?

JavaScript (JS) – це мова програмування, яка додає інтерактивність до веб-сторінок. Завдяки JS можна створювати динамічний контент, реагувати на дії користувача та маніпулювати HTML-елементами.

Способи підключення JavaScript

Існує три способи додавання JS-коду до HTML:

  1. Вбудований (inline) – код прописується безпосередньо в атрибуті onclick або іншому обробнику подій. 
    2. <button onclick="alert('Привіт, світ!')">Натисни мене</button>

    Результат:

  2. Внутрішній (internal) – код розміщується у <script> всередині HTML-документа. 
    3. <script>
  alert("Привіт, світ!");
</script>
  3. Зовнішній (external) – код пишеться в окремому файлі .js і підключається через <script>. Важливо знати, якщо потрібно залишити <script> у <head>, то слід додати атрибут defer, щоб код виконався після завантаження всього HTML документа. 
    4. <head>
  <script src="script.js" defer></script>
</head>

    Файл script.js:

    alert("Привіт, світ!");

Основні можливості JavaScript

JS використовується для:
✅ Роботи з HTML-елементами (зміна тексту, стилів, класів)
✅ Обробки подій (натискання кнопок, рух миші)
✅ Виконання обчислень і логічних операцій

Змінні в JavaScript

Змінні використовуються для збереження даних. Вони оголошуються за допомогою let, const або var: 

let ім’я = "Олексій";
const вік = 25;
var місто = "Київ";

Можна змінювати значення? Так Так Ні

Коли що використовувати?

  • Використовуйте const, якщо значення не змінюється.
  • Використовуйте let, якщо потрібно змінювати значення.
  • Не використовуйте var, оскільки він має недоліки, пов'язані з глобальною областю видимості та підняттям.

Взаємодія з користувачем

alert("Привіт!");                          // Виведе сповіщення  
console.log("Це повідомлення у консолі");  // Виведе текст у консоль  
let ім’я = prompt("Як тебе звати?");       // Запитає ім’я користувача

Функція alert() у JavaScript використовується для виведення простого діалогового вікна з повідомленням для користувача. Це модальне вікно, яке блокує взаємодію з іншими частинами веб-сторінки, поки користувач не закриє його, натиснувши "ОК".

Функція console.log() у JavaScript використовується для виведення інформації в консоль браузера або середовища виконання (наприклад, Node.js). Це один із найпоширеніших способів відлагодження коду.

Функція prompt() у JavaScript використовується для виведення діалогового вікна, яке запитує введення даних від користувача. Вона повертає рядок, введений користувачем, або null, якщо користувач натисне "Скасувати" (Cancel).

Основи DOM

Що таке DOM?

DOM (Document Object Model) – це об'єктна модель, яка представляє структуру веб-сторінки як деревоподібну структуру, де кожен елемент (тег, атрибут, текст) є об'єктом. За допомогою JavaScript можна отримувати доступ до елементів сторінки та змінювати їх.

Основні методи роботи з DOM

  1. Отримання елементів
    • getElementById(id) – отримує елемент за його унікальним ID. 
      • let element = document.getElementById("myElement");
    • getElementsByClassName(className) – отримує елементи за класом. 
      • let elements = document.getElementsByClassName("myClass");
    • getElementsByTagName(tagName) – отримує елементи за тегом. 
      • let elements = document.getElementsByTagName("p");
  2. Зміна елементів
    • innerHTML – змінює вміст елемента. 
      • let element = document.getElementById("myElement");
element.innerHTML = "Новий текст!";
    • style – змінює стиль елемента. 
      • let element = document.getElementById("myElement");
element.style.color = "red";
element.style.fontSize = "20px";
  3. Додавання та видалення елементів
    • createElement(tagName) – створює новий елемент. 
      • let newElement = document.createElement("p");
newElement.innerHTML = "Це новий абзац!";
document.body.appendChild(newElement);
    • removeChild(childElement) – видаляє елемент. 
      • let element = document.getElementById("myElement");
document.body.removeChild(element);

document.getElementById("addButton").addEventListener("click", function() {
    let inputText = document.getElementById("inputText").value;  // Отримуємо текст з input
    if (inputText !== "") {  // Перевірка на пусте значення

        document.getElementById("inputText").value = "";  // Очищаємо поле вводу
    }
});

Масиви в JavaScript

🔹 Що таке масив?

Масив — це колекція елементів, де кожен елемент має свій індекс. Масиви дозволяють зберігати кілька значень в одній змінній.

Основні операції з масивами:
  1. Створення масиву:
    2. let fruits = ["яблуко", "банан", "апельсин"];
  2. Доступ до елементів масиву:
    3. console.log(fruits[0]);  // виведе "яблуко"
console.log(fruits[2]);  // виведе "апельсин"
  3. Додавання елементів до масиву:
    • push() додає елемент в кінець масиву:
    fruits.push("груша");
console.log(fruits);  // ["яблуко", "банан", "апельсин", "груша"]
  4. Видалення елементів з масиву:
    • pop() видаляє останній елемент:
    fruits.pop();
console.log(fruits);  // ["яблуко", "банан", "апельсин"]
  5. Перебір елементів масиву:
    • forEach() дозволяє пройти по кожному елементу масиву:
    fruits.forEach(function(fruit) {
  console.log(fruit);
});
  6. Зміна елемента масиву:
    7. fruits[1] = "киві";
  console.log(fruits);  // ["яблуко", "киві", "апельсин"]
Завдання:
  1. Створи масив з трьох імен (наприклад, "Оля", "Іван", "Сергій").
  2. За допомогою push() додай ще одне ім'я.
  3. Використовуй forEach(), щоб вивести всі імена в консоль.
  4. Замініш друге ім'я на інше.
  5. Видалиш останнє ім'я за допомогою pop().
Ось правильний код: 
let array = ["Оля", "Іван", "Сергій"];
array.push("Михайло");

console.log("===== Список до змін =====");
array.forEach(function(name) {
    console.log(name);
});

// Змінюємо другий елемент
array[1] = "Андрій";

// Видаляємо останній елемент
array.pop();

console.log("===== Список після змін =====");
array.forEach(function(name) {
    console.log(name);
});

Додаткові методи масивів, які потрібно знати

  1. push(element) – додає елемент в кінець масиву.
  2. pop() – видаляє останній елемент.
  3. shift() – видаляє перший елемент.
  4. unshift(element) – додає елемент на початок.
  5. slice(start, end) – повертає частину масиву.
  6. splice(index, deleteCount, ...items) – змінює масив (додає або видаляє елементи).
  7. indexOf(element) – повертає індекс елемента або -1, якщо його немає.
  8. includes(element) – перевіряє, чи є елемент у масиві (true/false).
  9. reverse() – змінює порядок елементів на зворотний.
  10. sort() – сортує масив.

Методи перебору масивів

Окрім forEach(), є інші зручні методи:

  1. map() – створює новий масив, перетворюючи кожен елемент.
  2. filter() – повертає новий масив з елементами, що відповідають умові.
  3. find() – шукає перший елемент, що відповідає умові.
  4. some() – перевіряє, чи хоча б один елемент відповідає умові.
  5. every() – перевіряє, чи всі елементи відповідають умові.
  6. reduce() – обчислює одне значення, використовуючи всі елементи.
Приклад коду: 
let numbers = [1, 2, 3, 4, 5, 6];

// map() - множимо кожне число на 2
let doubled = numbers.map(num => num * 2);
console.log("map:", doubled); // [2, 4, 6, 8, 10, 12]

// filter() - залишаємо лише парні числа
let evenNumbers = numbers.filter(num => num % 2 === 0);
console.log("filter:", evenNumbers); // [2, 4, 6]

// find() - знаходимо перше число більше 3
let firstLarge = numbers.find(num => num > 3);
console.log("find:", firstLarge); // 4

// some() - чи є хоча б одне число більше 5?
let hasBigNumber = numbers.some(num => num > 5);
console.log("some:", hasBigNumber); // true

// every() - чи всі числа більше 0?
let allPositive = numbers.every(num => num > 0);
console.log("every:", allPositive); // true

// reduce() - сума всіх чисел
let sum = numbers.reduce((acc, num) => acc + num, 0);
console.log("reduce:", sum); // 21

Цикли в JavaScript

🔹 Що таке цикли?

Цикли в JavaScript дозволяють повторювати певний блок коду кілька разів. Це корисно, коли потрібно обробити велику кількість даних або виконати повторювані операції.

Основні типи циклів у JavaScript
  1. for – Використовується, коли відома точна кількість ітерацій.
  2. while – Виконується, поки певна умова є істинною.
  3. do...while – Виконується хоча б один раз, а потім продовжує виконання, поки умова істинна.
  4. for...of – Перебирає елементи масиву або іншої колекції.
  5. for...in – Використовується для перебору властивостей об'єкта.
Приклад циклу for 
for (let i = 0; i < 5; i++) {
  console.log("Ітерація №" + (i + 1));
}
👉 Виведе: 
Ітерація №1  
Ітерація №2  
Ітерація №3  
Ітерація №4  
Ітерація №5
Приклад циклу while 
let num = 0;
while (num < 3) {
  console.log("Число: " + num);
  num++;
}
👉 Виведе: 
Число: 0  
Число: 1  
Число: 2
✅ Твоє перше завдання по темі:
  1. Створи цикл for, який виведе числа від 1 до 10 в консоль.
    2. Правильний код:

🔹 Хочеш ускладнити завдання?

Напиши цикл do...while, який працює так само: віднімає 2 від x, але з обов’язковим першим виконанням коду, навіть якщо x відразу менше 0

  console.log(x); //Виводимо значення x
  x -= 2;         //Віднімаємо 2
} while (x > 0);
