Стилі HTML - CSS

CSS розшифровується як каскадні таблиці стилів.
CSS економить багато роботи. Він може керувати макетом кількох веб-сторінок одночасно.

Що таке CSS?

Каскадні таблиці стилів (CSS) використовуються для форматування макета веб-сторінки.
За допомогою CSS ви можете керувати кольором, шрифтом, розміром тексту, інтервалом між елементами, розташуванням і компонуванням елементів, фоновими зображеннями чи фоновими кольорами, різними дисплеями для різних пристроїв і розмірами екрану та багато іншого!

Використання CSS

CSS можна додати до документів HTML трьома способами:

  • Inline - за допомогою style атрибута всередині елементів HTML
  • Внутрішній - за допомогою <style> елемента в <head> розділі
  • Зовнішній - за допомогою <link> елемента для посилання на зовнішній файл CSS

Найпоширенішим способом додавання CSS є збереження стилів у зовнішніх файлах CSS.

Вбудований CSS

Вбудований CSS використовується для застосування унікального стилю до одного елемента HTML.

Внутрішній CSS

Внутрішній CSS використовується для визначення стилю для однієї сторінки HTML.

Зовнішній CSS

Зовнішня таблиця стилів використовується для визначення стилю для багатьох сторінок HTML.

На зовнішні таблиці стилів можна посилатися за допомогою повної URL-адреси або шляху відносно поточної веб-сторінки.

У цьому прикладі використовується повна URL-адреса для посилання на таблицю стилів:

<link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">

Цей приклад посилається на таблицю стилів, розташовану в папці html на поточному веб-сайті:

<link rel="stylesheet" href="/html/styles.css">

Цей приклад посилається на таблицю стилів, розташовану в тій же папці, що й поточна сторінка:

<link rel="stylesheet" href="styles.css">

Короткий зміст глави

  • Використовуйте атрибут HTML style для вбудованого стилю
  • Використовуйте елемент HTML <style> для визначення внутрішнього CSS
  • Використовуйте елемент HTML <link> для посилання на зовнішній файл CSS
  • Використовуйте елемент HTML <head> для зберігання елементів <style> і <link>
  • Використовуйте властивість CSS color для кольорів тексту
  • Використовуйте властивість CSS font-family для текстових шрифтів
  • Використовуйте властивість CSS font-size для розмірів тексту
  • Використовуйте властивість CSS border для меж
  • Використовуйте властивість CSS padding для простору всередині рамки
  • Використовуйте властивість CSS margin для простору за межами
