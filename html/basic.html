Основи HTML

У цьому розділі ми розберемо деякі основні приклади HTML.

Документи HTML

Всі HTML-документи повинні починатися з оголошення типу документа: <!DOCTYPE html>.
Сам HTML-документ починається з <html> і закінчується </html>.
Видима частина HTML-документа знаходиться між <body> і </body>.

HTML Заголовки

HTML заголовки визначаються за допомогою тегів від <h1> до <h6>. Тегом <h1> визначаються найбільш важливі заголовки. Тегом <h6> визначаються найменш важливі заголовки.

HTML Параграфи

HTML параграфи визначаються за допомогою тега <p>
