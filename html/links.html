HTML Посилання

Посилання можна знайти майже на всіх веб-сторінках. Посилання дозволяють користувачам переходити зі сторінки на сторінку.

Посилання HTML - Синтаксис

HTML тег <a> визначає гіперпосилання. Він має такий синтаксис:

Приклад 

<a href="url">Текст посилання</a>

Найважливішим атрибутом елемента <a> є атрибут href, який вказує пункт призначення посилання.

Текст посилання - це частина, яку буде видно читачеві.

Натиснувши на текст посилання, читач перенаправить за вказаною URL-адресою.

За умовчанням у всіх браузерах посилання відображатимуться так:

  • Невідвідане посилання - підкреслене і синім кольором
  • Відвідане посилання - підкреслене і фіолетовим кольором
  • Активне посилання - підкреслене і червоним кольором

Ви можете змінити кольори за замовчуванням, використовуючи CSS:

Приклад 

<style>
a:link {
  color: green;
  background-color: transparent;
  text-decoration: none;
}

a:visited {
  color: pink;
  background-color: transparent;
 text-decoration: none;
}

a:hover {
  color: red;
  background-color: transparent;
  text-decoration: underline;
}

a:active {
  color: yellow;
  background-color: transparent;
 text-decoration: underline;
}
</style>

Посилання часто стилізуються у вигляді кнопок за допомогою CSS:

Приклад 

<style>
a:link, a:visited {
  background-color: #f44336;
  color: white;
  padding: 15px 25px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
}

a:hover, a:active {
  background-color: red;
}
</style>

HTML-посилання – атрибут target

За замовчуванням пов’язана сторінка відображатиметься в поточному вікні браузера. Щоб змінити це, ви повинні вказати іншу ціль для посилання.

Цільовий атрибут target вказує, де відкривати пов’язаний документ.

Цільовий атрибут target може мати одне з таких значень:

  • _self - за замовчування, відкриває документ у тому самому вікні/вкладці, де його клацнули
  • _blank – відкриває документ у новому вікні чи вкладці
  • _parent - відкриває документ у батьківському фреймі
  • _top - відкриває документ у повному тілі вікна
