HTML Цитати і елементи цитування

У цьому розділі ми розглянемо наступні HTML елементи: <blockquote>, <q>, <abbr>, <address>, <cite> та <bdo>.

  • HTML тег <blockquote> - визначає розділ, цитований з іншого джерела.
  • HTML тег <q> - визначає коротку цитату.
  • HTML тег <abbr> - визначає абревіатуру або акронім, як-от «HTML», «CSS», «Mr.», «Dr.», «ASAP», «ATM».

Порада. Використовуйте глобальний атрибут title, щоб показати опис абревіатури/акроніму, коли ви наводите курсор миші на елемент.

  • HTML тег <address> визначає контактну інформацію автора/власника документа чи статті.
  • HTML тег <cite> визначає назву творчої роботи (наприклад, книга, вірш, пісня, фільм, картина, скульптура тощо).
  • HTML тег <bdo> використовується для зміни поточного напрямку тексту.
