Введення в HTML

HTML є стандартною мовою розмітки для створення веб-сторінок.

Що таке HTML?

  • HTML означає мову гіпертекстової розмітки
  • HTML є стандартною мовою розмітки для створення веб-сторінок
  • HTML описує структуру веб-сторінки
  • HTML складається з ряду елементів
  • Елементи HTML повідомляють браузеру, як відображати вміст
  • Елементи HTML позначають частини вмісту, наприклад «це заголовок», «це абзац», «це посилання» тощо.

Простий HTML-документ

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Заголовок сторінки</h1>
    <p>Важлива інформація.</p>
  </body>
</html>

Пояснення прикладу

  • Оголошення <!DOCTYPE html> визначає, що цей документ є документом HTML5
  • Елемент <html> є кореневим елементом сторінки HTML
  • Елемент <head> містить метаінформацію про HTML-сторінку
  • Елемент <title> визначає заголовок для сторінки HTML (який відображається в рядку заголовка браузера або на вкладці сторінки)
  • Елемент <body> визначає тіло документа та є контейнером для всього видимого вмісту, такого як заголовки, абзаци, зображення, гіперпосилання, таблиці, списки тощо.
  • Елемент <h1> визначає великий заголовок
  • Елемент <p> визначає абзац

Структура сторінки HTML

Усі документи HTML мають починатися з оголошення типу документа: <!DOCTYPE html>.

Сам документ HTML починається з <html> і закінчується на </html>.

Видима частина документа HTML знаходиться між <body> та </body>.

Нижче наведено візуалізацію структури HTML-сторінки:

<!DOCTYPE html>
<html>
<head>
<title>Назва веб сторінки</title>
</head>
<body>

<h1>Заголовок сторінки</h1>

<p>Важлива інформація</p>

</body>
</html>

Декларація <!DOCTYPE>

Оголошення <!DOCTYPE> представляє тип документа та допомагає браузерам правильно відображати веб-сторінки.
Він має з’являтися лише один раз у верхній частині сторінки (перед будь-якими тегами HTML).
У декларації <!DOCTYPE> не враховується регістр.
Оголошення <!DOCTYPE> для HTML5 таке:

<!DOCTYPE html>

Заголовки HTML

Заголовки HTML визначаються тегами <h1> ... <h6>.
<h1> визначає найважливіший заголовок, <h6> визначає найменш важливий заголовок:

Приклад <h1>Заголовок рівня 1</h1>
<h2>Заголовок рівня 2</h2>
<h3>Заголовок рівня 3</h3>
<h4>Заголовок рівня 4</h4>
<h5>Заголовок рівня 5</h5>
<h6>Заголовок рівня 6</h6>

Абзаци HTML

Абзаци HTML визначаються тегом <p>:

Приклад <p>Параграф для інфомації.</p>
<p>Інший параграф для інформації.</p>

Посилання HTML

HTML-посилання визначаються тегом <a>:

Приклад <a href="https://www.w3schools.com">Посилання на w3schools<a>

Пункт призначення посилання вказується в href атрибуті.

Зображення HTML

Зображення HTML визначаються тегом <img>.
Вихідний файл src, альтернативний текст alt, ширина width і висота height надаються як атрибути:

Приклад <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
