HTML Стилі

HTML атрибут style використовується для додавання стилів до елемента, таких як колір, шрифт, розмір тощо.

HTML атрибут style має наступний синтаксис:

Приклад 

<tagname style="property:value;">

property є властивістю CSS. value є значенням CSS.

  • CSS властивість background-color визначає колір фону для елемента HTML
  • CSS властивість color визначає колір тексту для елемент HTML
  • CSS властивість font-family визначає шрифт, який буде використовуватися для елемента HTML
  • CSS властивість font-size визначає розмір тексту для елемент HTML
  • CSS властивість text-align визначає горизонтальне вирівнювання тексту для елемента HTML
