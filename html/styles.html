Стилі HTML

Атрибут HTML style використовується для додавання стилів до елемента, таких як колір, шрифт, розмір тощо.

Короткий зміст глави

  • Використовуйте style атрибут для стилізації елементів HTML
  • Використовуйте background-color для кольору фону
  • Використовуйте color для кольорів тексту
  • Використовуйте font-family для текстових шрифтів
  • Використовуйте font-size для розмірів тексту
  • Використовуйте text-align для вирівнювання тексту
