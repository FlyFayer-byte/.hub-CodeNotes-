Введення в HTML

HTML – це стандартна мова розмітки для створення веб-сторінок.

Що таке HTML?

  • HTML розшифровується як Hyper Text Markup Language (мова гіпертекстової розмітки)
  • HTML є стандартною мовою розмітки для створення веб-сторінок
  • HTML описує структуру веб-сторінки
  • HTML складається з ряду елементів
  • Елементи HTML повідомляють браузеру, як відображати контент
  • Елементи HTML позначають фрагменти контенту, такі як "це заголовок", "це є абзацом", "це посилання" і т.д.

Простий HTML-документ

Приклад 

<!DOCTYPE html>
<html>
  <head>
    <title>Заголовок сторінки</title>
  </head>
  <body>
    <h1>Мій перший заголовок</h1>
    <p>Мій перший абзац.</p>
  </body>
</html>

Пояснення прикладу

  • Оголошення <!DOCTYPE html> визначає, що цей документ є документом HTML5
  • Елемент <html> є кореневим елементом сторінки HTML
  • Елемент <head> містить метаінформацію про HTML-сторінку
  • Елемент <title> визначає заголовок для сторінки HTML (який відображається в рядку заголовка браузера або на вкладці сторінки)
  • Елемент <body> визначає тіло документа та є контейнером для всього видимого вмісту, такого як заголовки, абзаци, зображення, гіперпосилання, таблиці, списки тощо.
  • Елемент <h1> визначає великий заголовок
  • Елемент <p> визначає абзац
