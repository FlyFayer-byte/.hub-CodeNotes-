HTML Атрибути

HTML Атрибути надають додаткову інформацію про елементи HTML.

Атрибути HTML

  • Всі елементи HTML можуть мати атрибути
  • Атрибути надають додаткову інформацію про елементи
  • Атрибути завжди вказуються в початковому тегу
  • Атрибути зазвичай подаються у парах ім'я/значення, як: name="value"

Атрибут href

Тег <a> визначає гіперпосилання, атрибут href вказує його URL-адресу.

Приклад 

<a href="https://hubcodenotes.com/">Це посилання</a>

Атрибут src

Тег <img> використовується для вбудовування зображення на HTML-сторінці. Атрибут src вказує шлях до зображення, яке буде відображено.

Приклад 

<img src="img_girl.jpg">

Вказати URL в атрибуті src можна двома способами:

1. Абсолютний URL - посилання на зовнішнє зображення, яке розміщується на іншому сайті.
Приклад: src="https://www.w3schools.com/images/img_girl.jpg".

Примітки: Зовнішні зображення можуть бути захищені авторським правом. Якщо ви використовуєте їх не отримавши дозвіл на це, ви можете порушити закони про авторські права. Крім того, ви не можете керувати зовнішнім зображенням, так як його можуть раптово видалити або змінити.

2. Відносна URL-адреса - посилання на зображення, яке розміщено всередині веб-сайту. Тут URL-адреса не включає доменне ім'я.
Якщо URL-адреса починається без косої риски вона буде відносно поточної сторінки.
Приклад: src="img_girl.jpg".
Якщо URL-адреса починається з косої риски, вона буде відносною до домену.
Приклад: src="/images/img_girl.jpg".

Підказка: Майже завжди краще використовувати відносні URL-адреси. Вони не зламаються при зміні домену.

Атрибути width і height

Тег <img> також повинен містити атрибути width і height, які вказують ширину і висоту зображення (у пікселях).

Приклад 

<img src="img_girl.jpg" width="500" height="600">

Атрибут alt

Атрибут alt для тегу <img> визначає альтернативний текст для зображення, якщо зображення з якоїсь причини не може бути відображено. Це може бути через повільне з’єднання, помилку в атрибуті src або якщо користувач використовує програму зчитування з екрана.

Приклад 

<img src="../images/img_big_dog.jpg" alt="Велика собака">

Атрибут style

Атрибут style використовується для додавання стилів до HTML елементів, таких як колір, шрифт, розмір тощо.

Приклад 

<p style="color:red;">Це абзац написаний червоним кольором.</p>

Атрибут lang

Ви завжди повинні включати атрибут lang всередині тега , щоб оголосити мову веб-сторінки. Це призначено для допомоги пошуковим системам і браузерам.

У наведеному нижче прикладі вказано англійську як мову сторінки:

Приклад 

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

Коди країн також можна додати до коду мови в атрибуті lang. Отже, перші два символи визначають мову HTML-сторінки, останні два визначають країну.

У наведеному нижче прикладі англійська як мова вказана, а Сполучені Штати як Країна:

Приклад 

<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

Ви можете побачити всі коди мов у довіднику HTML Language Code.
