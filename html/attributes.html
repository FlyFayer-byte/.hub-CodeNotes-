Атрибути HTML

Атрибути HTML надають додаткову інформацію про елементи HTML.

  • Усі елементи HTML можуть мати атрибути
  • Атрибути надають додаткову інформацію про елементи
  • Атрибути завжди вказуються в початковому тегу
  • Атрибути зазвичай складаються з пар ім’я/значення, наприклад: ім’я="значення"

Атрибути href

Тег <a> визначає гіперпосилання. Атрибут href визначає URL-адресу сторінки, на яку веде посилання:

Приклад <a href="https://www.w3schools.com">Відвідайте W3Schools</a>

Атрибут src

Тег <img> використовується для вставлення зображення на сторінку HTML. Атрибут src визначає шлях до зображення, яке буде показано:

Приклад <img src="img_girl.jpg">

Існує два способи вказати URL-адресу в src атрибуті:

1. Абсолютна URL-адреса – посилання на зовнішнє зображення, розміщене на іншому веб-сайті. Приклад:
src="https://www.w3schools.com/images/img_girl.jpg".
Примітки. Зовнішні зображення можуть бути захищені авторським правом. Якщо ви не отримаєте дозвіл на його використання, ви можете порушити закони про авторські права. Крім того, ви не можете контролювати зовнішні зображення; його можуть раптово видалити або змінити.
2. Відносна URL-адреса – посилання на зображення, розміщене на веб-сайті. Тут URL-адреса не містить доменного імені. Якщо URL-адреса починається без скісної риски, вона буде відносною поточної сторінки. Приклад:
src="img_girl.jpg".
Якщо URL-адреса починається з скісної риски, вона буде відносною домену. Приклад:
src="/images/img_girl.jpg".
Порада. Майже завжди краще використовувати відносні URL-адреси. Вони не зламаються, якщо ви зміните домен.

Атрибут alt

Необхідний alt атрибут для <img> тегу вказує альтернативний текст для зображення, якщо зображення з якоїсь причини не може бути відображено. Це може бути через повільне з’єднання, помилку в src атрибуті або якщо користувач використовує програму зчитування з екрана.

Атрибут стилю

Атрибут style використовується для додавання стилів до елемента, таких як колір, шрифт, розмір тощо.

Атрибут lang

Ви завжди повинні включати lang атрибут у <html> тег, щоб оголосити мову веб-сторінки. Це призначено для допомоги пошуковим системам і браузерам.

Коди країн також можна додати до коду мови в lang атрибуті. Отже, перші два символи визначають мову сторінки HTML, а останні два символи визначають країну.

Атрибут заголовка

Атрибут title визначає додаткову інформацію про елемент.

Короткий зміст глави

  • Усі елементи HTML можуть мати атрибути
  • Атрибут href визначає <a> URL-адресу сторінки, на яку веде посилання
  • Атрибут src визначає <img> шлях до зображення, яке буде показано
  • Атрибут width і height надають <img> інформацію про розмір зображень
  • Атрибут alt надає <img> альтернативний текст для зображення
  • Атрибут style використовується для додавання стилів до елемента, таких як колір, шрифт, розмір тощо
  • Атрибут lang тегу <html> оголошує мову веб-сторінки
  • Атрибут title визначає додаткову інформацію про елемент
