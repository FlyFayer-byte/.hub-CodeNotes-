HTML Атрибути

HTML Атрибути надають додаткову інформацію про елементи HTML.

Атрибути HTML

  • Всі елементи HTML можуть мати атрибути
  • Атрибути надають додаткову інформацію про елементи
  • Атрибути завжди вказуються в початковому тегу
  • Атрибути зазвичай подаються у парах ім'я/значення, як: name="value"

Атрибут href

Тег <a> визначає гіперпосилання, атрибут href вказує його URL-адресу.

Приклад 

<a href="https://hubcodenotes.com/">Це посилання</a>

Атрибут src

Тег <img> використовується для вбудовування зображення на HTML-сторінці. Атрибут src вказує шлях до зображення, яке буде відображено.

Приклад 

<img src="img_girl.jpg">
