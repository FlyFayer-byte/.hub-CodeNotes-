HTML Графіка Canvas

Елемент HTML <canvas> використовується для малювання графіки на веб-сторінці.

Графіка вище, створена за допомогою <canvas>. Він показує чотири елементи: червоний прямокутник, градієнтний прямокутник, багатоколірний прямокутник і багатоколірний текст.

Що таке HTML Canvas?

HTML Елемент <canvas> використовується для малювання графіки на льоту за допомогою JavaScript.

Елемент <canvas> є лише контейнером для графіки. Ви повинні використовувати JavaScript, щоб фактично намалювати графіку.

Canvas має кілька методів малювання контурів, прямокутників, кіл, тексту та додавання зображень.

Canvas підтримується всіма основними браузерами.

Приклади Canvas

Canvas — це прямокутна область на HTML сторінці. За замовчуванням полотно не має рамок і вмісту.

Розмітка виглядає так: HTML 
 
  <canvas id="myCanvas" width="200" height="100" style="border: 1px solid black;"></canvas>

Примітка. Завжди вказуйте атрибут id (на який потрібно посилатися в сценарії), а також атрибути width і height для визначення розміру полотна. Щоб додати рамку, використовуйте атрибут style.

Додайте JavaScript

Після створення прямокутної області полотна ви повинні додати JavaScript, щоб виконати малюнок. Нижче наведено декілька прикладів:

Намалюйте лінію: HTML 
 
<canvas id="draw-line" width="200" height="100" style="border: 1px solid black;"></canvas>
<script>
  var c = document.getElementById("draw-line");
  var ctx = c.getContext("2d");
  ctx.moveTo(0, 0);
  ctx.lineTo(200, 100);
  ctx.stroke();
</script>

Малювання лінійного градієнта: HTML 
 
<canvas id="draw-linear-gradient" width="200" height="100" style="border: 1px solid black;"></canvas>
<script>
  var c = document.getElementById("draw-linear-gradient");
  var ctx = c.getContext("2d");

  // Create gradient
  var grd = ctx.createLinearGradient(0, 0, 200, 0);
  grd.addColorStop(0, "red");
  grd.addColorStop(1, "white");

  // Fill with gradient
  ctx.fillStyle = grd;
  ctx.fillRect(10, 10, 150, 80);
</script>
Результат

Малювання кругового градієнта: HTML

  grd.addColorStop(1, "white");
