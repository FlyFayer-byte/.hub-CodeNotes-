Цитата HTML і елементи цитування

У цьому розділі ми розглянемо елементи <blockquote>, <q>, <abbr>, <address>, <cite> і <bdo>.

Tag Description
<abbr> Визначає абревіатуру або акронім, як-от «HTML», «CSS», «Mr.», «Dr.», «ASAP», «ATM»
<address> Визначає контактну інформацію для автора/власника документа чи статті
<bdo> Використовується для зміни поточного напрямку тексту
<blockquote> Визначає розділ, цитований з іншого джерела
<cite> Визначає назву творчої роботи (наприклад, книга, вірш, пісня, фільм, картина, скульптура тощо)
<q> Визначає коротку вбудовану цитату
