HTML Стилі - CSS

CSS розшифровується як Cascading Style Sheets - каскадні таблиці стилів.

CSS економить багато роботи. Він може керувати макетом кількох веб-сторінок одночасно.

Що таке CSS?

Каскадні таблиці стилів (CSS) використовуються для форматування макета веб-сторінки.

За допомогою CSS ви можете керувати кольором, шрифтом, розміром тексту, інтервалом між елементами, розташуванням і компонуванням елементів, фоновими зображеннями або фоновими кольорами, різними дисплеями для різних пристроїв і розмірами екрану, і багато іншого!

Порада. Слово «каскадний» означає, що стиль, застосований до батьківського елемента, також застосовуватиметься до всіх дочірніх елементів у цьому батьківському. Отже, якщо ви встановите колір основного тексту на «синій», усі заголовки, абзаци та інші текстові елементи всередині основної частини також отримають той самий колір (якщо ви не вкажете щось інше)!

Використання CSS

CSS можна додати в HTML документи 3 способами:

  • Inline (вбудований або рядковий) - за допомогою атрибута style всередині елементів HTML
  • Internal (внутрішній) - за допомогою елемента <style> в розділі <head>
  • External (зовнішній) - за допомогою елемента <link> для посилання на зовнішній CSS-файл

Найпоширенішим способом додавання CSS є збереження стилів у зовнішніх файлах CSS.

Inline (вбудований) CSS стиль

Вбудований CSS використовується для застосування унікального стилю до одного елемента HTML.

Вбудований CSS використовує атрибут style елемента HTML.

У наступному прикладі колір тексту елемента <h1> встановлюється синім, а колір тексту елемента <p> - червоним:

Приклад 

<h1 style="color:blue;">Текст заголовка синього кольору</h1>
<p style="color:red;">Текст параграфу синього кольору</p>

Internal (внутрішній) CSS стиль

Внутрішній CSS використовується для визначення стилю для однієї сторінки HTML.

Внутрішній CSS визначається в розділі <head> сторінки HTML в елементі <style>.

У наступному прикладі колір тексту ВСІХ елементів <h1> (на цій сторінці) встановлюється синім, а колір тексту ВСІХ елементів <p> - червоним. Крім того, сторінка відображатиметься з фоновим кольором <code> "powderblue":

Приклад 

<!DOCTYPE html>
<html>
<head>
  <style>
    body {background-color: powderblue;}
    h1   {color: blue;}
    p    {color: red;}
  </style>
</head>
<body>
  <h1>Це заголовок</h1>
  <p>Це параграф</p>
</body>
</html>

External (зовнішній) CSS стиль

Зовнішня таблиця стилів використовується для визначення стилю для багатьох сторінок HTML.

Щоб використовувати зовнішню таблицю стилів, додайте посилання на неї в розділ <head> кожної HTML-сторінки:

Приклад 

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <h1>Це заголовок</h1>
  <p>Це параграф</p>
</body>
</html>

Зовнішню таблицю стилів можна написати в будь-якому текстовому редакторі. Файл не має містити HTML-код і має бути збережений із розширенням .css.

Ось як виглядає файл styles.css:

Приклад 

body {
  background-color: powderblue;
}
h1 {
  color: blue;
}
p {
  color: red;
}

Примітка: Детальніше про текстові редактори ви можете прочитати в розділі HTML Редактори на сайті W3Schools українською.
