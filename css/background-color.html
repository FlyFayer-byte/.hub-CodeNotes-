CSS Фони Background

CSS властивості фону background використовуються для додавання фонових ефектів для елементів.

У цих розділах ви дізнаєтеся про такі CSS властивості фону:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background (скорочена властивість)

CSS background-color - Колір фону

Властивість background-color визначає колір фону елемента.

Приклад: CSS

Колір фону сторінки встановлюється так:

 
body {
  background-color: lightblue;
}

У CSS, колір найчастіше визначається як:

  • дійсна назва кольору, наприклад "red"
  • шістнадцяткове значення, наприклад "#ff0000"
  • значення RGB, наприклад "rgb(255,0,0)"

Інші елементи

Ви можете встановити колір фону для будь-яких елементів HTML:

Приклад: CSS

Тут елементи <h1>, <p> і <div> матимуть різні фонові кольори:

 
h1 {
  background-color: green;
}
p {
  background-color: lightblue;
}
div {
  background-color: yellow;
}

Opacity / Transparency - Непрозорість / Прозорість

Властивість opacity визначає непрозорість/прозорість елемента. Він може приймати значення від 0.0 до 1.0. Чим менше значення, тим прозоріше:

opacity 1
opacity 0.6
opacity 0.3
opacity 0.1
Приклад: CSS 
 
div {
  background-color: green;
  opacity: 0.3;
}

Примітка. У разі використання властивості opacity для додавання прозорості фону елемента, всі його дочірні елементи успадковують однакову прозорість. Це може ускладнити читання тексту всередині повністю прозорого елемента.

Прозорість з використанням RGBA

Якщо ви не хочете застосовувати непрозорість до дочірніх елементів, як у нашому прикладі вище, використовуйте значення кольорів RGBA. У наступному прикладі встановлюється непрозорість для кольору фону, а не тексту:

100% opacity
60% opacity
30% opacity
10% opacity
Приклад: CSS 
 
div {
  background: rgba(0, 128, 0, 0.3) /* Green background with 30% opacity */
}

CSS Background Image - Фонове зображення

Властивість background-image визначає зображення для використання в якості фону елемента.

За замовчуванням зображення повторюється, тому воно покриває весь елемент.

Приклад: CSS

Установіть фонове зображення для сторінки:

 
body {
  background-image: url("paper.gif");
}

Фонове зображення також можна встановити для визначених елементів, таких як елемент <p> (для окремого параграфу):

Приклад: CSS

У цьому прикладі показано невдале поєднання тексту та фонового зображення. Текст важко читається:

 
p {
  background-image: url("paper.gif");
}

CSS Background Repeat - Повторення фону

CSS background-repeat

За замовчуванням властивість background-image повторює зображення як горизонтально, так і вертикально.

Деякі зображення слід повторювати тільки по горизонталі або вертикалі, інакше вони можуть виглядати дивно

Приклад: CSS 
 
body {
  background-image: url("../images/gradient_bg.png");
}

Якщо зображення повторюється тільки по горизонталі background-repeat: repeat-x;, фон буде виглядати краще

Приклад: CSS 
 
body {
  background-image: url("../images/gradient_bg.png");
  background-repeat: repeat-x;
}

Порада: Щоб повторити зображення по вертикалі, встановіть background-repeat: repeat-y;

CSS background-repeat: no-repeat

Відображення фонового зображення тільки один раз також визначається властивістю background-repeat:

Приклад: CSS

Показати фонове зображення тільки один раз:

 
body {
  background-image: url("../images/img_tree.png");
  background-repeat: no-repeat;
}

CSS background-position

Властивість background-position використовується для вказування положення (позиції) фонового зображення.

Приклад: CSS

Розмістіть фонове зображення в правому верхньому куті:

 
body {
 background-image: url("../images/img_tree.png");
 background-repeat: no-repeat;
 background-position: right top;
}

CSS Background Attachment - Прикріплення фону

CSS background-attachment

Властивість background-attachment визначає, чи повинно фонове зображення прокручуватись або фіксуватися (не буде прокручуватись разом з іншою частиною сторінки):

Приклад: CSS

У цьому прикладі фонове зображення буде фіксованим:

 
body {
  background-image: url("../images/img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: fixed;
}
Приклад: CSS

У цьому прикладі фонове зображення має прокручуватись разом з іншою частиною сторінки:

 
body {
  background-image: url("../images/img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: scroll;
}

CSS Background Shorthand - Скорочена властивість

Щоб скоротити код, також можна вказати всі властивості фону в одній єдиній властивості. Вона називається скороченою властивістю.

Замість того, щоб писати:

Приклад: CSS 
 
body {
  background-color: #fff;
  background-image: url("img_tree.png");
  background-repeat: no-repeat;
 background-position: right top;
}

Ви можете використовувати скорочену властивість background:

Приклад: CSS 
 
body {
  background: #fff url("../images/img_tree.png") no-repeat right top;
}

При використанні скороченої властивості порядок значень властивостей наступний:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

Немає значення, якщо одне зі значень властивостей відсутнє, доки інші знаходяться в такому порядку. Зауважте, що ми не використовуємо властивість background-attachment у наведених вище прикладах, оскільки вона не має значення.

Усі властивості фону CSS

Властивість Опис
background Встановлює всі властивості фону в одній об’яві
background-attachment Встановлює, чи є фонове зображення фіксованим, чи прокручується разом з іншою частиною сторінки
background-clip Визначає область малювання фону
background-color Встановлює колір фону елемента
background-image Встановлює фонове зображення для елемента
background-origin Визначає, де розташовані фонові зображення.
background-position Встановлює початкову позицію фонового зображення
background-repeat Встановлює, як буде повторюватись фонове зображення
background-size Визначає розмір фонового зображення (зображень)
