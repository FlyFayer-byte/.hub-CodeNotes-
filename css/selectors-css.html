CSS Селектори

CSS селектори використовуються для "пошуку" (або вибору) HTML-елементів, які ви хочете стилізувати.

Ми можемо розділити CSS-селектори на п'ять категорій:

  • Прості селектори (вибір елементів на основі імені, ідентифікатора, класу)
  • Селектори комбінаторів (вибір елементів на основі певного зв’язку між ними)
  • Селектори псевдокласів (вибір елементів на основі певного стану)
  • Селектори псевдоелементів (вибір і стиль частини елемента)
  • Селектори атрибутів (вибір елементів на основі атрибута або значення атрибута)

На цій сторінці ми розповімо про найосновніші CSS селектори.

Селектор елементів CSS

Селектор елементів вибирає елементи HTML на основі назви елемента.

Приклад: CSS

Тут всі елементи <p> на сторінці будуть вирівняний по центру, з червоним кольором тексту:

 
p {
  text-align: center;
  color: red;
}

Селектор ідентифікатора CSS

Селектор ідентифікатора використовує атрибут id елемента HTML для вибору певного елемента.

Ідентифікатор елемента є унікальним на сторінці, тому селектор ідентифікатора використовується для вибору одного унікального елемента!

Щоб вибрати елемент із певним ідентифікатором, напишіть символ решетки #, а потім ідентифікатор елемента.

Приклад: CSS

Наведене нижче правило CSS буде застосовано до елемента HTML з id="para1":

 
#para1 {
  text-align: center;
  color: red;
}

CSS Селектор класів

Селектор вибирає елементи HTML із певним атрибутом class>.

Щоб вибрати елементи з певним класом, напишіть крапку (.), а потім назву класу.

Приклад: CSS

В цьому прикладі всі HTML елементи з class="center" будуть червоними та вирівняні по центру:

 
.center {
  text-align: center;
  color: red;
}

Ви також можете вказати, що клас має зачіпати лише визначені HTML елементи.

Приклад: CSS

В цьому прикладі лише елементи <p> з class="center" будуть вирівняні по центру:

 
p.center {
  text-align: center;
  color: red;
}

HTML елементи також можуть посилатись на кілька класів.

Приклад: HTML

В цьому прикладі елемент <p> буде стилізовано у відповідності з class="center" та class="large":

 
<p class="center large">Цей параграф реферер для двох класів.</p>

Універсальний CSS селектор

Селектор групування вибирає всі елементи HTML з однаковими визначеннями стилю.

Подивіться на наступний код CSS (елементи h1, h2 і p мають однакові визначення стилю):

Приклад: CSS 
 
h1 {
  text-align: center;
  color: red;
}
h2 {
  text-align: center;
  color: red;
}
p {
  text-align: center;
  color: red;
}

Краще буде згрупувати селектори, щоб згорнути код.

Щоб згрупувати селектори, розділіть їх комою.

Приклад: CSS

У цьому прикладі ми згрупували селектори з коду вище:

 
h1, h2, p {
  text-align: center;
  color: red;
}

Всі прості CSS селектори

Селектор Приклад Опис прикладу
.class .intro Обирає всі елементи з class="intro"
#id #firstname Обирає елемент з id="firstname"
* * Обирає всі елементи на сторінці
element p Обирає всі елементи <p>
element,element,.. div, p Обирає всі елементи <div> та всі елементи <p>
