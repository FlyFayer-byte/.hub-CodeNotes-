Введення в CSS

CSS — це мова, яку ми використовуємо для оформлення веб-сторінки.

Що таке CSS?

  • CSS розшифровується як Каскадні Таблиці Стилів (Cascading Style Sheets)
  • CSS описує, як елементи HTML повинні відображатися на екрані, паперу, або на інших носіях
  • CSS економить багато роботи. Він може керувати компонуванням Кілька веб-сторінок одночасно
  • Зовнішні таблиці стилів зберігаються у файлах CSS

Навіщо використовувати CSS?

CSS використовується для визначення стилів для ваших веб-сторінок, включаючи дизайн, макет а також варіації дисплея для різних пристроїв і розмірів екранів.

CSS вирішив велику проблему

HTML НІКОЛИ не призначався для того, щоб містити теги для форматування веб-сторінки!

HTML був створено для опису вмісту веб-сторінки, наприклад:

  • <h1>Це заголовок</h1>
  • <p>Це абзац.</p>

CSS економить багато роботи!

Визначення стилів зазвичай зберігаються у зовнішніх файлах .css.

За допомогою зовнішнього файлу таблиці стилів ви можете змінити вигляд цілого веб-сайту, змінивши лише один файл!
