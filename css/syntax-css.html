CSS Синтаксис

Правило CSS складається з селектора та блоку оголошення.

Селектор (selector) вказує на елемент HTML, який потрібно стилізувати.

Блок декларацій містить одну або кілька декларацій, розділених крапкою з комою.

Кожне оголошення (declaration) містить назву (property) властивості CSS і значення (value), розділені двокрапкою.

Кілька декларацій CSS розділені крапкою з комою, а блоки декларацій оточені фігурними дужками.

У наступному прикладі всі елементи <p> будуть вирівняні по центру з червоним кольором тексту:

Приклад: CSS 
 
p {
  color: red;
  text-align: center;
}

Пояснення прикладу

  • p — це селектор у CSS (він вказує на елемент HTML, який потрібно стилізувати: <p>).
  • code — це властивість, а red — це значення властивості
  • text-align — це властивість, а center — це значення властивості
