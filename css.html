Основи CSS

Що таке CSS?

CSS (Cascading Style Sheets) – це мова стилів, яка використовується для оформлення веб-сторінок. Вона дозволяє змінювати кольори, шрифти, розміщення елементів та інші візуальні аспекти.

Способи підключення CSS

Існує три способи додавання стилів до HTML:

  1. Вбудований стиль (inline) – стилі прописуються безпосередньо в атрибуті style HTML-елемента. 
    2. <p style="color: blue; font-size: 20px;">Цей текст синій і має розмір 20px.</p>
  2. Внутрішній стиль (internal) – стилі розміщуються у тегу <style> в <head>. 
    3. <head>
  <style>
    p {
      color: green;
      font-size: 18px;
    }
  </style>
</head>
  3. Зовнішній стиль (external) – стилі пишуться в окремому файлі .css та підключаються через <link>. 
    4. <head>
  <link rel="stylesheet" href="styles.css">
</head>

    Файл styles.css:

    p {
  color: red;
  font-size: 16px;
}

Основні властивості CSS

  • Колір тексту: color: red;
  • Розмір шрифту: font-size: 20px;
  • Фон: background-color: yellow;
  • Вирівнювання тексту: text-align: center;

    • Відступи:

  • margin – це зовнішній відступ (відстань між елементами).
  • padding – це внутрішній відступ (відстань між вмістом елемента і його межами).

  • Ширина та висота: width, height
  • Границі: border: 2px solid black;
